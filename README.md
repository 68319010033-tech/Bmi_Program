# 🏋️‍♂️ BMI & Health Risk Calculator

โปรแกรมคำนวณดัชนีมวลกาย (Body Mass Index - BMI) และประเมินความเสี่ยงต่อโรค อ้างอิงตามเกณฑ์ภูมิภาคเอเชียแปซิฟิก (Asian Academic Criteria) พัฒนาด้วย JavaScript (Node.js) พร้อมระบบตรวจสอบความถูกต้องของข้อมูล (Input Validation) และ Unit Test

## ✨ คุณสมบัติระบบ (Features)
* **คุมเข้มความปลอดภัยข้อมูล (Input Validation):** ดักจับกรณีผู้ใช้กรอกข้อความ/ตัวหนังสือ, ค่าติดลบ หรือค่าที่มากเกินมาตรฐานมนุษย์ปกติ (น้ำหนัก > 500 กก. หรือ ส่วนสูง > 250 ซม.)
* **แม่นยำตามเกณฑ์เอเชีย:** แบ่งเกณฑ์ดัชนีมวลกายออกเป็น 5 ระดับ เพื่อความเหมาะสมกับสรีระคนเอเชีย
* **ประเมินความเสี่ยงอัจฉริยะ:** นอกจากการจำแนกกลุ่ม ยังสามารถประเมินระดับอันตรายและความเสี่ยงต่อสุขภาพได้ทันที
* **มี Unit Test ครอบคลุม:** มั่นใจในความถูกต้องด้วยชุดทดสอบคุณภาพสูง (Jest) กว่า 18 Cases

## 📊 เกณฑ์การวัดผล (Metrics & Standards)

| ค่า BMI | ระดับความอ้วน (Category) | ความเสี่ยงต่อโรค (Health Risk) |
| :--- | :--- | :--- |
| **น้อยกว่า 18.5** | น้ำหนักน้อย / ผอม | มากกว่าคนปกติ (เสี่ยงขาดสารอาหาร) |
| **18.5 - 22.9** | ปกติ (สุขภาพดี) | เท่าคนปกติ |
| **23.0 - 24.9** | ท้วม / โรคอ้วนระดับ 1 | อันตรายระดับ 1 |
| **25.0 - 29.9** | อ้วน / โรคอ้วนระดับ 2 | อันตรายระดับ 2 |
| **30.0 ขึ้นไป** | อ้วนมาก / โรคอ้วนระดับ 3 | อันตรายระดับ 3 |
