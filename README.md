# 🏋️‍♂️ BMI & Health Risk Calculator

โปรแกรมคำนวณดัชนีมวลกาย (BMI) และประเมินความเสี่ยงต่อโรค อ้างอิงตามเกณฑ์ภูมิภาคเอเชียแปซิฟิก พัฒนาด้วย JavaScript (Node.js) 

---

## 🚀 การติดตั้ง (Installation)

1. เปิด Terminal ในโฟลเดอร์โปรเจกต์ (`Bmi_Program`)
2. รันคำสั่งด้านล่างเพื่อติดตั้งระบบทดสอบ (Jest):
```bash
npm install
